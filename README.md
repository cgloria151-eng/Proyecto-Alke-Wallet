Proyecto Alke Wallet

Este proyecto consiste en una aplicación web básica que simula una billetera digital.

Funcionalidades
- Inicio de sesión
- Envío de dinero
- Navegación entre páginas

Tecnologías utilizadas
- HTML
- CSS
- JavaScript

Descripción
La aplicación permite al usuario ingresar mediante un login simulado y realizar el envío de dinero a un destinatario.
